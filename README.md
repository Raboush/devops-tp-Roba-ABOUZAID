
# API de Gestion des Étudiants

## Description

API REST simple permettant de gérer des étudiants. Ce projet est réalisé dans le cadre du module DevOps afin de mettre en pratique les bonnes pratiques de développement, de gestion de code, de tests, d'intégration continue et de conteneurisation.
API REST permettant de gérer les informations des étudiants d'un établissement scolaire.

---

## Stack

* Node.js
* Express.js
* Git & GitHub
* GitHub Actions
* Docker
* Docker Compose
* PostgreSQL
* Jest
* ESLint
* Prettier

---

## Lancer le projet

Les étapes d'installation et d'exécution seront complétées au fur et à mesure de l'avancement du projet.
